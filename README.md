# DP-SGD-VRD

# 1. Running Environment
Install all packages as specified in `requirements.txt`.


# 2. To Reproduce
For the data required to run the tasks on `IMDB` dataset, it is located at  `./datasets/nltk`, the `GloVe` embedding for words are used. 

The modified norm layer and resnets can be found at `./datasets/norm_layer.py` and `./datasets/RESNETS.py`.

Setup is adjustable in script `run.sh`.

Our running log is located at `./logs`
